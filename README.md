# BeEnergy ⚡

**Marketplace Web3 para la tokenización y comercialización de energía renovable en comunidades.**

BeEnergy permite a comunidades pequeñas que ya cuentan con infraestructura solar o eólica gestionar y vender su remanente energético de forma transparente, mediante blockchain Stellar.

---

## ¿Qué problema resuelve?

Las comunidades con paneles solares o fuentes eólicas generan más energía de la que consumen, sin herramientas para comercializarla de forma directa y transparente. BeEnergy tokeniza ese excedente y habilita un marketplace peer-to-peer dentro de la propia comunidad.

---

## Funcionalidades

- ⚡ Tokenización de kWh como HDROP tokens en Stellar blockchain
- 💱 Marketplace P2P para compra y venta de energía excedente
- 📊 Gestión de producción y consumo en tiempo real
- 💰 Rendimientos automáticos con DeFi (DeFindex)
- 🔒 Rankings con privacidad mediante Zero-Knowledge Proofs

---

## Stack

- **Frontend:** React · v0
- **Backend:** Node.js · Soroban Smart Contracts
- **Blockchain:** Stellar SDK · HDROP Token · Freighter Wallet
- **Base de datos:** Supabase
